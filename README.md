# Kimia Yaourt - Site Vitrine

Ce projet est le site vitrine officiel de **Kimia Yaourt**. Il a pour but de présenter l'entreprise, ses différentes gammes de yaourts et de permettre aux clients de prendre contact.

---

## 🚀 Fonctionnalités
- **Page d'accueil :** Présentation de la marque et de son histoire.
- **Catalogue :** Affichage des produits et des saveurs disponibles.
- **Formulaire de contact :** Un moyen simple et sécurisé pour nous joindre, protégé contre le spam.
- **Responsive Design :** Entièrement optimisé pour une lecture parfaite sur smartphone et tablette.

## 🛠️ Technologies utilisées
- **HTML5** / **CSS3** / **JavaScript**

## 💻 Comment lancer le projet en local
1. Clonez ce dépôt.
2. Ouvrez le fichier `index.html` dans votre navigateur.
